💰 Daily Expense Tracker (Java)

📌 Overview

This is a simple command-line based Expense Tracker built using Java. It helps users record and manage their daily expenses efficiently.

🎯 Features

* Add expenses (amount, category, note)
* View all expenses
* Calculate total spending
* Simple and user-friendly interface

🛠️ Technologies Used

* Java
* IntelliJ IDEA
* ArrayList (Data Structure)

⚙️ Setup Instructions

1. Install Java

Download and install JDK (Java Development Kit 17 or above)

2. Clone Repository

```bash
git clone https://github.com/your-username/expense-tracker-java
cd expense-tracker-java
```

3. Compile the Code

```bash
javac Expense.java ExpenseTracker.java
```

4. Run the Application

```bash
java ExpenseTracker
```

▶️ Usage

1. Choose option from menu
2. Add expense details
3. View expenses or total spending
